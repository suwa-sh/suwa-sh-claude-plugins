# 貸出状況を確認する - フロントエンド仕様

## 変更概要

司書ポータルに貸出状況一覧画面を実装する。全利用者の貸出中書籍をページネーション付きで一覧表示する。

## 画面仕様

### 貸出状況一覧画面

- **URL**: /admin/loans
- **アクセス権**: 司書ロール
- **ポータル**: admin

#### 表示要素とコンポーネントマッピング

| 要素 | 種別 | デザインシステムコンポーネント | 説明 |
|------|------|------------------------------|------|
| 貸出一覧 | テーブル | LoanRecord (default/overdue) | 延滞中はoverdue variant |
| 貸出状態バッジ | 表示 | BookLoanStatusBadge | 貸出中/延滞中 |
| ページネーション | ナビ | - | オフセットベース 20件/ページ |

#### デザイントークン参照

| 用途 | トークン | 値 |
|------|---------|---|
| テーブルヘッダ背景 | var(--component-table-header-bg) | #F8FAFC |
| 行高さ | var(--component-table-row-height) | 3rem |

#### UIロジック

- **状態管理**: page, perPage, totalCount を state 管理。ページ切替で再取得
- **バリデーション**: なし（参照のみ）
- **ローディング**: 初回読み込み時は Skeleton UI（テーブル行形状）
- **エラーハンドリング**: API エラー時はエラーバナー

#### 操作フロー

1. 司書がサイドバーの「貸出管理」>「貸出状況一覧」をクリック
2. GET /api/v1/loans?page=1&per_page=20 で一覧取得
3. LoanRecord で貸出一覧表示（延滞中は赤色ハイライト）
4. ページネーションボタンでページ切替
