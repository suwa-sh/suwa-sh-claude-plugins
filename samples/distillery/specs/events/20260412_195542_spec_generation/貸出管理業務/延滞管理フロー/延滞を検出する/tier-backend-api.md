# 延滞を検出する - バックエンドAPI仕様

## 変更概要

延滞を検出するの API エンドポイントを実装する。

## API 仕様

### 延滞一覧取得

- **メソッド**: GET
- **パス**: /api/v1/loans/overdue
- **認証**: OAuth2/OIDC (司書ロール必須)


### 延滞検出バッチトリガー

- **メソッド**: POST
- **パス**: /api/v1/loans/detect-overdue
- **認証**: OAuth2/OIDC (司書ロール必須)



#### エラーレスポンス

| ステータスコード | 条件 | レスポンス |
|----------------|------|-----------|
| 400 | バリデーションエラー | RFC 7807 Problem Details |
| 401 | 未認証 | RFC 7807: title="Unauthorized" |
| 403 | 権限なし | RFC 7807: title="Forbidden" |
| 404 | リソース未存在 | RFC 7807: detail="リソースが見つかりません" |
