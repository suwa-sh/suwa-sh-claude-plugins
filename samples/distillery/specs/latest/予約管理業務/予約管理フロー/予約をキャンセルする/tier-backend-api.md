# 予約をキャンセルする - バックエンドAPI仕様

## 変更概要

予約をキャンセルするの API エンドポイントを実装する。

## API 仕様

### 予約キャンセル

- **メソッド**: DELETE
- **パス**: /api/v1/reservations/{id}
- **認証**: OAuth2/OIDC (利用者ロール)



#### エラーレスポンス

| ステータスコード | 条件 | レスポンス |
|----------------|------|-----------|
| 400 | バリデーションエラー | RFC 7807 Problem Details |
| 401 | 未認証 | RFC 7807: title="Unauthorized" |
| 403 | 権限なし | RFC 7807: title="Forbidden" |
| 404 | リソース未存在 | RFC 7807: detail="リソースが見つかりません" |
