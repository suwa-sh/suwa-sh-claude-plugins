# phase2：業務
- 「0_RDRAZeroOne/phase1/初期要望分析.md」を読み初期の要望を理解する
- 「0_RDRAZeroOne/phase1/業務.tsv」を読み初期要望の業務を理解する
- 「0_RDRAZeroOne/phase1/要求.tsv」を読み機能要求/非機能要求を理解する
- 「0_RDRAZeroOne/phase1/ビジネスポリシー.tsv」を読み初期要望のビジネスポリシーを理解する
- 「0_RDRAZeroOne/phase1/アクター.tsv」を読みアクターの役割を理解する
- 「0_RDRAZeroOne/phase1/状態.tsv」を読み初期要望の状態モデルを理解する
- 上記ファイルから業務を再整理し「業務.tsv」を出力する

# 背景理解
- phase1のアクターと業務は初期要望から個別に洗い出している
- 「phase1/業務」をベースに「phase2/業務」を作成する

## 業務の洗出し
- 「phase1/業務.tsv」と「phase2/業務.tsv」で同じ項目はそのまま出力する
- 「phase1/業務」の説明を参考に「業務/BUC」をブレークダウンし「アクティビティ」と「UC」を追加する
  - BUCの複数アクティビティが業務フローを実現する
- アクティビティは仕事の単位となる
  - 要求に「アクティビティ」と「UC」に関わるものがあれば、その要求を実現するために必要な「アクティビティ」と「UC」ものを洗い出す
    - 具体的な仕事やシステムとの関係が機能要求、非機能要求に記述されている場合は、その要求を満たす、「アクティビティ」「UC」を出力する
- UCは仕事でコンピュータを使う仕事の単位となる
- 「0_RDRAZeroOne/phase1/初期要望分析.md」に業務フローの要素になるものがあれば、アクティビティとして追記する
- 「0_RDRAZeroOne/phase1/初期要望分析.md」にシステムの機能について記述があれば、UCとして追記する
- 「状態.tsv」の状態モデル内の状態の遷移はUCによって遷移される
- 状態モデルに関わるBUCを識別し、状態を遷移させるUCは業務内のUCに対応する

## アクティビティ/UCへのブレークダウン例
- 価値受益アクターと価値提供アクターをアクティビティ/UC毎に分割する
- UCは価値受益アクターか価値提供アクターのどちらかのアクターと結びつく
  - 価値受益アクターが設定されているときは価値提供アクターカラムは「""」を設定
  - 価提供値アクターが設定されているときは価値受益アクターカラムは「""」を設定
- アクティビティとUCはつながるアクターの役割に応じた内容になる
  - アクティビティ/UCが行うことは関わるアクターがBUCのアクターの役割に応じたは顧客などの

### phase1「業務.tsv」
業務/BUC/価値受益アクター/価値提供アクター/
--------------------------
業務A/BUC_A/価値受益アクターA/価値提供アクターA,価値提供アクターB

### phase2「業務.tsv」
業務/BUC/価値受益アクター/価値提供アクター/アクティビティ/UC
--------------------------
業務A/BUC_A/価値受益アクターA/""/アクティビティA/UC_A
業務A/BUC_A/""/価値提供アクターA/アクティビティB/UC_B
業務A/BUC_A/""/価値提供アクターB/アクティビティC/UC_C

### 例の説明
- phase1「業務.tsv」で定義された一つの価値受益アクターと二つの価値提供アクターは３件のレコードに分かれて定義される
- 分割した３レコードのアクティビティとUCは役割が変わるので名前が変わる
- 価値受益アクターが設定された時は価値提供アクターが""をセット
- 価値提供アクターが設定された時は価値受益アクターが""をセット

## 出力ルール
- 出力ファイル：「0_RDRAZeroOne/phase2/業務.tsv」
- 出力には必ずヘッダー（カラム名）行を出力する

### 「業務」のフォーマット
出力形式：TSV（タブ区切り）
カラム構成：
1列目: 業務：業務を分類する名前
2列目: BUC：洗い出した「BUC」の名前。
3列目: 価値受益アクター：BUCから価値を得る一つのアクター
4列目: 価値提供アクター：BUCの価値を提供するために関わる複数のアクター
5列目: アクティビティ：仕事の単位となる
6列目: UC：ユースケース（UC）アクティビティで使うシステムを表す
7列目: 説明：UCの内容を説明
ヘッダー行を含め、各フィールドはタブ文字で区切ってください。
